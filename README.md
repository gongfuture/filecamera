巡查相机（文件相机）
一款纯粹、公益、无广告的开源水印相机，致力于为一线工作人员提供清爽、高效的拍照与管理体验。

介绍
“巡查相机”是一款开源的水印相机应用，最低支持安卓10（SDK版本 34）。本项目采用前端WebView与后端Java相结合的混合开发模式，实现了功能灵活与性能稳定的平衡。

核心特色之一是引入了文件夹式管理系统，让您能够像在电脑上管理文件一样，轻松整理和归类不同项目、不同阶段的照片，告别杂乱无章的相册。

此外，应用提供了高度可配置的水印，您可以根据需求自由定制水印内容。照片还可生成导航二维码，方便后续的溯源与定位。

为了提供更丰富的功能，我们集成了高德Web服务的API接口。只要您拥有一个API Key，即可解锁天气信息与地理位置逆编码（将经纬度转换为具体的地点名称）功能，让您的照片信息更加完整、专业。

更进一步，通过与我们的另一个开源项目 路线管理助手 联动，您可以直接导入geojson格式的路线数据，实现路线自动匹配与桩号精准计算，极大提升了工程巡查、道路维护等工作的效率。

项目缘起

在日常工作中我们发现，许多一线工作人员（如建筑工人、巡检员等）经常需要使用水印相机记录工作内容。然而，市面上的同类应用充斥着大量广告，甚至存在误导性消费陷阱。这不仅影响了工作效率，还可能给他们带来不必要的经济损失。

为了解决这一痛点，我们开发了“巡查相机”。这是一款完全公益性质的应用，我们承诺无任何广告、无任何隐藏付费项目。我们的目标很简单：提供一款干净、可靠、易用的工具，让每一位使用者都能安心、高效地完成工作。

软件架构
前端: WebView，负责UI展示与用户交互，具备良好的跨平台潜力与灵活性。
后端: Java (Android原生)，负责处理相机调用、图像处理、文件管理、API通信等核心逻辑，保证应用的性能与稳定性。
开发模式: 采用成熟的混合开发模式，兼顾了开发效率与用户体验。
安装教程
直接安装 (推荐):

下载最新的 apk 压缩包。
解压压缩包。
找到 .apk 文件并点击安装即可。
源码编译:

确保您的开发环境已配置 Android SDK 34。
克隆或下载本仓库源码。
使用 Android Studio 打开项目并编译生成APK。
使用说明
开启高级地理功能:

前往高德开放平台申请一个Web服务的API密钥 (Key)。
在App的设置界面中，找到相应选项并填入您的密钥。
成功配置后，水印将可以自动添加天气与详细的地理位置信息。
实现路线匹配与桩号计算:

访问我们的配套项目: 路线管理助手。
使用该项目生成或导出您需要的 geojson 路线文件。
在“巡查相机”App中，进入“路线管理”功能，导入您刚刚生成的 geojson 文件。
导入成功后，在拍照时App即可根据您的GPS位置自动匹配路线并计算桩号。
